# Simple Android Messenger

Демонстрационный Android-проект простого мессенджера на Kotlin + Jetpack Compose.

## Что реализовано
- Экран входа (`LoginScreen`) с логином/паролем.
- Защищённое хранение учётных данных пользователя через `EncryptedSharedPreferences`.
- Загрузка списка чатов (`ChatListScreen`).
- Экран чата с подгрузкой новых сообщений (периодический polling).
- Отправка тестового сообщения в выбранный чат.
- Выход из аккаунта с очисткой сохранённых учётных данных.

## Запуск
1. Откройте проект в Android Studio (Giraffe+ / Hedgehog+).
2. Дождитесь синхронизации Gradle.
3. Запустите конфигурацию `app` на эмуляторе или устройстве.

## Технически
- Данные чатов и сообщений фейковые и хранятся в `FakeMessengerRepository`.
- Подгрузка новых сообщений имитируется таймером в `ChatViewModel`.
- Учётные данные сохраняются в `SecureCredentialsStorage`.
- Архитектура: простая связка `Repository + ViewModel + Compose UI`.
